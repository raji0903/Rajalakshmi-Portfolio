## Quick start (local)
1. Extract the ZIP.
2. Open `index.html` in your browser (double-click). For EmailJS to work, host the site on a web server (e.g., GitHub Pages or local server).
   - For a quick local server: `npx http-server` or `python -m http.server` inside the folder.
3. Replace images in `/assets` with your real photos, logos and screenshots.
4. Replace EmailJS IDs in `contact.html` and `js/main.js`:
   - In contact.html: call `emailjs.init('USER_ID');`
   - In js/main.js: replace `'SERVICE_ID'` and `'TEMPLATE_ID'` in `emailjs.sendForm` call.
